## Prerequisites
- Nextflow
- Apptainer or Docker

## Installation
- Clone the sf-tractomics repository

- Setup your environment:
    Choice 1: Use the devcontainer
    Choice 2: Install tools locally
        - Install nf-core or use the devcontainer
        - Install nf-test

## Running the pipeline locally

## Configuring the pipeline
- Editing the nextflow.config file
- Configuring the modules with `conf/modules/*.config`

## Building the schema
Building the schema `nf-core pipelines schema build`

## Testing your changes
To test your changes, run:
`nf-test test -profile=+docker`

To update the snapshots, run:
`nf-test test -profile=+docker --clean-snapshot --update-snapshot`
